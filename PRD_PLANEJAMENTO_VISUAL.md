# PRD — Planejamento Visual (Infinite Canvas)

**Foco:** Pensamento espacial, mapas conceituais e storyboards.

---

## Features Principais

### Feature 1 — Infinite Canvas (Modo Lousa)
**Objetivo:** Espaço livre para brainstorming e conexão de ideias.
- **Interface:** Zoom infinito, panning e ferramentas de desenho (formas, setas, notas adesivas).
- **Integração:** Possibilidade de arrastar notas `.md` para o canvas como "blocos de texto" vivos.
- **Persistência:** Formato `.canvas.json` salvo no workspace.

### Feature 2 — Virtualization & Performance
**Objetivo:** Garantir fluidez em telas complexas.
- **Mecânica:** Renderização seletiva baseada na viewport para suportar centenas de elementos sem queda de FPS.
