# PRD — Sistema de Plugins e Extensões (Add-ons)

**Visão Geral:**
Para manter o núcleo do editor performático e focado em texto, todas as funcionalidades de nicho (Acadêmicas, Desenvolvedor, Design e Estudo de Línguas) são estruturadas como Plugins.
